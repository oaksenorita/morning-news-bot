厳選ニュース解説

退職一時金を「廃止」する会社は増えるのか

ロジカル要約
  • 背景/経緯: 労働市場の流動化が進み、従来の「終身雇用・年功序列」を前提とした退職金制度が優秀な若手人材の確保の妨げになっている。
  • 事象: 退職一時金を廃止し、その分を月々の給与やDC（確定拠出年金）に上乗せする「給与シフト」を選択する企業が急増。
  • 結果/影響: 企業は固定費の不確実性を排除できる一方、従業員は自ら資産形成を行うリテラシーが求められる時代へ。
地銀視点の深掘り

北海道内の中小企業においても、事業承継や採用難を背景に賃金体系の見直しは急務です。取引先企業が退職金制度の改定を検討される際は、従業員向けの「資産形成セミナー（NISA/iDeCo等）」の実施をセットで提案し、行員としてのプレゼンスを高める絶好の機会となります。

第1次オイルショックと令和のオイルショックの比較

ロジカル要約
  • 背景/経緯: 地政学リスクの長期化と、脱炭素への過度な投資シフトにより、化石燃料の供給構造が脆弱になっている。
  • 事象: 1970年代の供給不足と比較し、現在は「資源ナショナリズム」に加え、精製能力の不足という多層的なリスクに直面。
  • 結果/影響: エネルギー価格の高止まりが常態化し、特に輸入依存度の高い日本経済にとって深刻なインフレ圧力に。
地銀視点の深掘り

エネルギーコストの上昇は、北海道の基幹産業である農業や物流、観光業に直撃します。融資先の決算書を見る際は、売上高だけでなく、エネルギー関連コストの推移を注視しましょう。省エネ設備導入のための設備投資資金（グリーンローン）の提案につなげるのが行員の腕の見せ所です。

円が大暴落、“1ドル360円時代”への懸念と政治不信

ロジカル要約
  • 背景/経緯: 国内外の金利差に加え、政治的空白や経済政策への信頼低下が円売りに拍車をかけている。
  • 事象: 一部の市場関係者から、戦後の固定相場制である1ドル360円水準まで購買力が低下する可能性が指摘される。
  • 結果/影響: 輸入価格の激騰による家計の圧迫と、海外資本による国内資産（土地・企業）の買い叩きが加速。
地銀視点の深掘り

円安はインバウンド需要には追い風ですが、原材料を輸入に頼る地元メーカーにとっては死活問題です。為替予約やデリバティブを用いたリスクヘッジの提案はもちろん、海外販路開拓を目指す顧客には、円安を逆手に取った輸出拡大のコンサルティングを検討しましょう。
