金融ニュース＆実践インサイト

親の遺産3000万円を「全部あげる」と兄が言い出した！ ありがたいけど「贈与税がかかるんじゃ？」と夫に言われて不安に…受け取る場合の税金はどれくらいになりますか？

ロジカル要約

  • 背景/経緯: 親の遺産3000万円の分割協議にあたり、兄弟間での譲渡申し出が発生。贈与税と相続税の区別について疑問が生じている。
  • 事象: 遺産分割協議の中で特定の相続人が全額を受け取る場合は「相続税」の対象となるが、決定後に個人間で資産を移動させた場合は「贈与税」が発生する可能性がある。
  • 結果/影響: 正しい手続きを経て遺産分割協議書を作成・調印すれば贈与税の発生を防ぐことができ、相続税の基礎控除枠の活用により税負担を抑えることが可能。

地銀視点の深掘り

北海道内でも高齢化に伴い、遺産相続や実家の不動産処分の話題は非常に多くなっています。お客様との会話で「身内でお金をまとめようと考えている」と伺った際は、手続きのタイミング次第で税金の種類が変わる注意点をお伝えすると信頼感に繋がります。当行の相続相談窓口や信託商品、提携する税理士へのトスアップ（連携）を意識したアプローチが有効です。

続きを読む

30代の息子が加入した生命保険を、私が毎月「2万円」払っています。夫から「贈与になるのでは？」と言われましたが、保険料を親が負担しているだけでも、税金がかかることはあるのでしょうか？

ロジカル要約

  • 背景/経緯: 親が30代の子の生命保険料（月2万円・年24万円）を継続的に代渡しているケースにおける課税上のリスク。
  • 事象: 保険料の年間支払い額自体は暦年贈与の基礎控除（110万円）内であるため支払い時の贈与税はかからないが、将来満期金や解約返戻金を受け取る際、「保険料負担者」と「受取人」の関係によって贈与税や所得税の対象となる。
  • 結果/影響: 保険契約の形式（契約者・被保険者・受取人・保険料負担者）を正しく管理しておかないと、将来的な受取時に予期せぬ課税が発生する。

地銀視点の深掘り

店頭窓口や訪問先で親世代・シニア層のお客様と話す際、お子様や孫世代への金銭的支援は非常に親しみやすい話題です。「良かれと思って払っている保険料」も受取時の関係性で税金が変わる点を話題にしつつ、暦年贈与を活用した資産承継や当行取扱の保険・年金商品の見直し提案へ繋げるきっかけとして活用できます。

60代からの老後資金づくりと資産防衛｜夫婦・単身世帯の貯蓄中央値と「先取り・資産分割・支出のコントロール」の具体策

ロジカル要約

  • 背景/経緯: 定年退職を迎える60代以降において、物価高（インフレ）や長寿化に伴う老後資金の枯渇リスクへの対応が急務となっている。
  • 事象: 貯蓄の中央値データを踏まえ、「先取り貯蓄」「資産の役割別分割（日常用・守る・増やす）」「支出の管理」という3つのアプローチで老後資産を守り育てる。
  • 結果/影響: 退職金等のまとまった資金を全額預金に放置せず、目的ごとに色分けして適切なリスク管理を行うことで、持続可能な老後生活を形成できる。

地銀視点の深掘り

退職金運用や再雇用期の資産管理は、地銀にとって最も重要な営業領域の一つです。お客様に対して「預金だけでなく資産を3つのグループ（使う・守る・増やす）に分けましょう」という考え方を提示することで、安全域を確保しながら一部を投資信託やNISAに案内するコンサルティング営業が実践できます。
